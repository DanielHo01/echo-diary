# 项目概述

## 产品愿景

**回响 Echo Diary** 是一款基于 AI 的个人日记应用，通过简单的事件记录和智能采访，帮助用户生成有温度的每日日记。

## 核心价值

- 让日记记录变得简单自然（语音/文字）
- 通过 AI 采访深度挖掘记忆
- 生成结构化、有情感的日记
- 支持历史回顾和情感分析

## 用户场景

| 用户类型 | 场景描述 | 核心需求 |
|---------|---------|---------|
| 日常用户 | 每天记录生活点滴 | 快速录入、AI 辅助生成 |
| 写作爱好者 | 想要更有文采的日记 | 多风格生成、可编辑 |
| 回忆记录者 | 想要回顾过去的自己 | 时间轴浏览、情感分析 |
| 多设备用户 | 在手机和电脑上都使用 | 数据导出/备份 |

## MVP 用户流程

```

    │
    ▼
┌─────────────┐
│  记录事件    │  ← 输入今天发生的事（文字/语音）

       │
       ▼
┌─────────────┐
│  AI 采访    │  ← 基于事件生成问题，用户回答

       │
       ▼

       ▼
```
